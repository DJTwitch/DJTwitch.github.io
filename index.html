About

This program will allow users watching live streams on twitch.tv
to vote on the live music they can listen to during the stream.
This application was created by Jacob Tatz, John Lim, Daniel D'Amico, and John DeFalco.

How To Use

All OS's

1. Download source code

2. Download necessary modules

3. Run or compile DJTwitch
